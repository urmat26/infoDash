# InfoDash — Информационная панель

Дэшборд мониторинга для офисного экрана/телевизора. Показывает в реальном времени погоду, курсы валют, новости и случайные факты. Данные обновляются автоматически без перезагрузки страницы.

## Стек

- HTML / CSS / JavaScript (vanilla)
- [Chart.js](https://www.chart.js) — графики
- [Vercel](https://vercel.com) — деплой

## API

| Виджет | API | Ключ |
|--------|-----|------|
| Погода | [Open-Meteo](https://open-meteo.com) | Не нужен |
| Курсы валют | [ExchangeRate API](https://open.er-api.com) | Не нужен |
| Новости | [Лента.ру](https://lenta.ru) + [Хабр](https://habr.com) через [RSS2JSON](https://api.rss2json.com) | Не нужен |
| Факты | [uselessfacts.jsph.pl](https://uselessfacts.jsph.pl) | Не нужен |

## Функции

- 🌤 Погода: температура, влажность, ветер, прогноз на 7 дней
- 💱 Курсы валют: USD/EUR/RUB к KGS
- 📰 Новости: три категории (Все / IT / Мир)
- 📊 График: линейный, столбчатый, круговая диаграмма
- ⏱ Часы реального времени с кольцом секунд
- 💡 Случайные факты (англ.) с кэшированием
- 🌓 Тёмная и светлая тема
- 📱 Адаптивная вёрстка (TV 1920×1080 → телефон)
- ⚠ Обработка ошибок + localStorage кэш
- 🔄 Автообновление каждые 60 секунд

## Структура

```
dashboard/
├── index.html           # Основная страница
├── style.css            # Стили
├── app.js               # Инициализация, обновление, факты
├── config.example.js    # Конфиг (скопируй в config.js)
├── chart-setup.js       # Chart.js графики
├── widgets/
│   ├── clock.js         # Часы
│   ├── weather.js       # Погода
│   ├── currency.js      # Курсы валют
│   └── news.js          # Новости
└── README.md
```

## Запуск

Просто открой `index.html` в браузере.

Для деплоя на Vercel — подключи репозиторий к [vercel.com](https://vercel.com) и выбери папку `dashboard`.

## Команда

Проект выполнен в рамках учебного курса. 3 урока × 4 часа = 12 часов.
